# UMDS
Universal Microcontroller Development System

UMDS is an universal Development System for different Microcontrollers

actually ther are

CPU boards for:
- ATMEGA (688, 328, xxUy)
- ESP32 (WROOM32, WROVER, NodeCpu (30pin or 38pin)
- PI-Pico

BUS boards:
- 7 slots (4E distance)
- 4 slots (4E distance)
